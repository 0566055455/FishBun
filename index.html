FishBun

  • Download Latest

Multi Image Picker library for Android that Easy to use and Easy to customize

Introduction

FishBun is a beautiful and easy image picker library!

FishBun.with(Your Activity).startAlbum();

You can easily implement a multi-image picker.

You can also make your apps more consistent with a variety of customization features.

Features

Easy

Easy to use 

FishBun.with(Your Activity).startAlbum();

FishBun.with(Your Fragment).startAlbum();//if you use in Fragment

Customize

You can easily create your own beautiful image picker.

- In Album

- In Picker

- Common
  - setActionBarColor(int actionbarColor)
  - setActionBarColor(int actionbarColor, int statusBarColor)
  - setActionBarColor(int actionbarColor, int statusBarColor, boolean isStatusBarLight)
  - setActionBarTitleColor(int actionbarTitleColor)
  - setActionBarTitle(String actionBarTitle)
  - textOnNothingSelected(String message)
  - textOnImageSelectionLimitReached(String message) 
  - setHomeAsUpIndicatorDrawable(Drawable icon)
  - setOkButtonDrawable(Drawable icon)

- Album
  - setAlbumThumbnailSize(int size)
  - setAlbumSpanCount(int portraitSpanCount, int landscapeSpanCount)
  - setAlbumSpanCountOnlyLandscape(int landscapeSpanCount)
  - setAlbumSpanCountOnlyPortrait(int portraitSpanCount)
  - setAllViewTitle(String allViewTitle)

- Picker
  - setPickerSpanCount(int spanCount)

Etc

Various other functions

- setMaxCount(int count)
- setMinCount(int count)
- setArrayPaths(ArrayList arrayPaths)
- setCamera(boolean isCamera)
- setRequestCode(int requestCode)
- setButtonInAlbumActivity(boolean isButton)
- setReachLimitAutomaticClose(boolean isAutomaticClose)
